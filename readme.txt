== Frequently Asked Questions ==

= Will the Blocks Bakery cause my website to slow down? =

Without a doubt! The Blocks Bakery for Gutenberg is designed with simplicity and speed in mind. It is lightweight and fast thanks to its module architecture and clean code. As a result, it will have no effect on the speed and performance of your website.


= Is it okay if I use the Blocks Bakery on a client's website? =

Yes, of course. The Blocks Bakery can be used on both your own and your clients' websites.

= Is it compatible with other block plugins? =

No doubt! Blocks Bakery can be used in conjunction with both the core and third-party block plugins.

= When I disable the plugin, what happens to my Blocks Bakery content? =

Your material will be presented in a simplified version; if you reactivate Blocks Bakery, it will revert to its original appearance.

== Installation ==

1. Upload the plugin to the /wp-content/plugins/ directory.
2. Activate the Blocks Bakery WordPress plugin through the 'Plugins' menu in WordPress.
3. Use Blocks Bakery on your next page or post.
